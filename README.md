# מצבת כוחות | Force Posture

לוח מודיעין צבאי בזמן אמת — השוואת כוחות בין ארה״ב, ישראל ואיראן.

Real-time military intelligence dashboard comparing force posture of USA, Israel, and Iran.

## תכונות | Features

- 📊 השוואת כוח אדם, ציוד וחימושים בין שלוש מדינות
- 📈 גרפים מודיעיניים אינטראקטיביים (Recharts)
- 🗺️ מפת טווחי טילים
- 🌐 תמיכה בעברית ואנגלית (RTL/LTR)
- 📱 עיצוב רספונסיבי למובייל ודסקטופ

## טכנולוגיות | Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Recharts

## מקורות | Sources

GlobalFirepower 2026, IISS, SIPRI, IDF, CENTCOM, CSIS
